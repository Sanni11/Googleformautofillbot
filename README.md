# Google Form Autofill Bot

A Python script that automatically fills out Google Forms using Selenium browser automation.

## Requirements

- Python 3.x
- [Selenium](https://pypi.org/project/selenium/)
- [ChromeDriver](https://chromedriver.chromium.org/downloads) (must match your installed Chrome version)

## Installation

```bash
pip install selenium
```

Download ChromeDriver and place it in your PATH or the same directory as the script.

## Usage

```bash
python autofill_script.py
```

Update the script with your target Google Form URL and the answers you want to submit before running.

## Notes

- Make sure your ChromeDriver version matches your installed version of Google Chrome.
- This script is intended for personal or testing use only.
